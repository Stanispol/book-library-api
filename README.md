# Мини-библиотека книг API

Практическое занятие №1  
Мини веб-служба и конвейер обработки запросов (Node.js + Express + TypeScript)

## О проекте

Простое REST API, которое хранит книги в памяти процесса.  
Реализован конвейер обработки запросов с middleware, единый формат ошибок и логирование с requestId.

**Эндпоинты:**
- `GET /api/items` — получить список всех книг  
- `GET /api/items/{id}` — получить книгу по идентификатору (404 если не найдена)  
- `POST /api/items` — создать новую книгу

**Правила валидации:**
- `title` — обязательное, строка, минимум 3 символа  
- `price` — обязательное, число ≥ 0

**Конвейер обработки запроса:**
1. Генерация requestId  
2. Логирование + замер времени выполнения  
3. Обработка ошибок (единый формат ответа)

## Как запустить

```bash
# 1. Клонировать репозиторий (если ещё не склонировал)
git clone https://github.com/Stanispol/book-library-api.git
cd book-library-api

# 2. Установить зависимости
npm install

# 3. Запустить в режиме разработки (с авто-перезапуском)
npm run dev
